How ID Verification Works

The ID must satisfy three conditions:

  1. Exactly 8 digits in length.
  2. All characters must be numeric digits (0-9).
  3. The dot product of the digits against descending weights [8, 7, 6, 5, 4, 3, 2, 1] must be divisible by 11.

Example: 11191031 → 1x8+1x7+1x6+9x5+1x4+0x3+3x2+1x1 = 66 → 66 / 11 = 6 (student)
